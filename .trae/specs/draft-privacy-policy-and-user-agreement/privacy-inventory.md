## 二、系统权限

| 权限 | 用途 | 是否敏感 |
|------|------|----------|
| INTERNET | 访问网络，加载线上 Web 资源、调用服务端接口 | 否 |
| ACCESS_NETWORK_STATE | 检测网络状态，优化请求策略 | 否 |
| ACCESS_WIFI_STATE | 检测 Wi-Fi 状态 | 否 |
| RECEIVE_USER_PRESENT | 接收用户解锁屏幕事件（极光推送依赖） | 是 |
| WAKE_LOCK | 保持设备唤醒，确保推送到达 | 是 |
| POST_NOTIFICATIONS | 发送推送通知（Android 13+ 需要单独申请） | 是 |
| JPUSH_MESSAGE | 极光推送自定义权限，用于接收推送消息 | 是 |
| VIBRATE | 推送到达时振动提醒 | 否 |

## 三、收集的个人信息

### 3.1 用户主动提供的信息

| 信息类型 | 收集场景 | 是否必填 |
|----------|----------|----------|
| 对话内容 | 用户在 AI 聊天面板输入的文本 | 是 |
| 思维导图内容 | 用户创建、编辑的节点、分支、文本 | 是 |
| 工作区名称 | 用户创建工作区时填写 | 否 |
| 文件内容 | 用户上传的文档（PDF、Word、TXT 等）用于 AI 解析 | 否 |
| 联系方式 | 用户提交反馈时填写的联系方式（选填） | 否 |

### 3.2 自动收集的信息

| 信息类型 | 收集目的 | 说明 |
|----------|----------|------|
| 访客 ID | 标识匿名用户 | 本地生成并存储于 localStorage，非真实设备 ID |
| 访客签名密钥 | 服务端接口鉴权 | 服务端生成，本地存储 |
| IP 地址 | 安全风控、限流、反滥用 | 服务端记录，最多保留最近 10 条历史 IP |
| UserAgent | 埋点分析、兼容性处理 | 浏览器/ WebView 标识 |
| 页面 URL | 埋点分析 | 记录用户当前页面 |
| 操作事件 | 产品优化、功能分析 | 节点创建、分支创建、模板使用、地图库操作等 |
| 设备型号 | 推送服务 | 极光 Registration ID 关联 |
| APP 版本 | 推送服务、版本分析 | 极光推送注册时上报 |
| Registration ID | 推送服务 | 极光 SDK 生成的设备级推送标识 |

## 四、第三方 SDK

| SDK 名称 | 服务商 | 用途 | 收集信息 |
|----------|--------|------|----------|
| 极光推送 JPush | 深圳市和讯华谷信息技术有限公司 | 向用户推送消息通知 | Registration ID、设备型号、APP 版本、操作系统版本、网络信息、位置信息（用于推送通道优化） |

## 五、数据存储

| 数据类型 | 存储位置 | 说明 |
|----------|----------|------|
| 用户创建的思维导图、对话记录、工作区信息 | 服务端 MongoDB 数据库 | 存储在运营方控制的服务器上 |
| 访客 ID、访客密钥、当前工作区 ID | 客户端 localStorage | 本地存储，不上传 |
| 用户自定义 AI 模型配置（含 API Key） | 客户端 localStorage | 本地存储，不上传到本应用服务器 |
| 上传的文档文件 | 服务端 uploads 目录 | 用于 AI 解析，需用户主动上传 |
| 埋点事件数据 | 服务端 MongoDB | 用于产品分析 |
| 推送设备信息 | 服务端 MongoDB user_devices 集合 | 用于极光推送 |

## 六、数据共享

- 用户自定义 AI 模型配置中的 API Key 仅用于客户端直接调用对应第三方 AI 服务商接口，不经过本应用服务器。
- 使用内置 AI 服务时，用户对话内容会发送至运营方服务端，由服务端调用第三方大模型 API 生成回复。
- 极光推送 SDK 会将必要的设备信息和推送标识传输至极光服务器，用于消息下发。

## 七、数据保留期限

- 思维导图、对话记录、工作区数据：自用户最后一次操作起保留至用户删除或账号（访客身份）注销为止。
- 埋点数据：保留不超过 12 个月，用于产品分析。
- IP 历史记录：最多保留最近 10 条。
- 反馈信息：保留至问题处理完成或法律法规要求的期限。

## 八、用户权利

当前版本已具备的能力：
- 用户可主动删除自己创建的思维导图文件和工作区。
- 用户可主动清除浏览器/应用本地存储（将丢失本地访客身份，服务端历史数据仍保留）。

当前版本暂缺的入口（建议在隐私政策中说明联系运营方处理）：
- 无自助「账号注销」入口。
- 无自助「数据导出」入口。
- 无自助「撤回同意」入口。

## 九、未成年人保护

- 本应用主要面向成年人用户。
- 若用户为未成年人，应在监护人指导下使用。
- 运营方不会主动收集未成年人的个人信息。

## 十、注意事项

- 当前 `client/android/local.properties` 中明文保存了 keystore 密码与极光 AppKey，存在安全隐患，建议后续改为环境变量或 CI Secret 注入。
- 用户自定义 AI API Key 存储在客户端 localStorage，若设备被他人使用或 localStorage 泄露，可能导致 Key 被盗用，应在用户协议中明确提示风险。
