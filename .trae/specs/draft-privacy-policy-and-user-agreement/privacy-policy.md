# DeepMindMap 隐私政策

> 版本号：v1.0
> 更新日期：2026-06-30
> 生效日期：2026-07-02

## 引言

DeepMindMap（以下简称"本应用"）由 **罗楚瑞**（以下简称"我们"或"运营方"）运营。我们非常重视用户的个人信息和隐私保护，致力于为用户提供一个安全、可信的对话式 AI 思维导图创作环境。本《隐私政策》将帮助用户了解我们如何收集、使用、存储、共享和保护用户的个人信息，以及用户享有的相关权利。

请用户在使用本应用前仔细阅读并充分理解本政策。一旦用户开始使用本应用，即表示用户已同意本政策所述内容。如用户不同意本政策，请停止使用本应用。

## 一、适用范围

本政策适用于 DeepMindMap 客户端（Android 应用、Web 应用）以及由运营方提供的服务端接口、推送服务、客服支持等相关服务。

本政策不适用于用户通过本应用访问的第三方服务（如用户自行配置的第三方 AI 服务商），这些服务适用其各自的隐私政策。

## 二、我们收集的信息

### 2.1 用户主动提供的信息

为向用户提供服务，用户在使用本应用时可能需要或自愿提供以下信息：

- **对话内容**：用户在 AI 聊天面板中输入的文本、指令、问题等。
- **思维导图内容**：用户创建、编辑、导入、导出的节点文本、分支结构、样式设置等。
- **工作区信息**：用户创建的工作区名称、成员关系、权限设置等。
- **上传文件内容**：用户主动上传的文档（如 PDF、Word、TXT、Markdown 等），用于 AI 解析和生成思维导图。
- **反馈信息**：用户通过反馈功能提交的问题描述、建议内容及选填的联系方式。

**请注意**：本应用目前不收集用户的敏感个人信息，包括但不限于身份证号、银行卡号、生物识别信息、通信录、短信、通话记录、精准位置信息等。

### 2.2 我们自动收集的信息

为提供服务、保障安全、优化体验，我们可能会自动收集以下信息：

- **访客标识信息**：包括本地生成的访客 ID（Visitor ID）和由服务端下发的访客签名密钥，用于识别用户身份和接口鉴权。
- **设备和网络信息**：包括 UserAgent、设备型号、操作系统版本、APP 版本、屏幕分辨率、网络类型、IP 地址等。
- **日志和操作信息**：包括页面访问记录、操作事件（如节点创建、分支扩展、模板使用、地图库访问等）、功能使用频次、错误日志等。
- **推送标识信息**：极光推送 SDK 生成的 Registration ID，用于向用户设备推送消息通知。

### 2.3 来自第三方的信息

当用户选择使用第三方 AI 服务商（如 OpenAI、智谱 AI 等）时，相关 API Key 由用户自行配置并存储在本地，我们不会收集或存储该等密钥。但用户使用内置 AI 服务时，其对话内容会经我们的服务端转发至对应的第三方大模型服务提供商。

### 2.4 系统权限

为保障本应用相关功能的实现与正常运行，我们可能会申请或使用以下系统权限：

| 权限名称 | 权限说明 | 使用目的 |
|---------|---------|---------|
| INTERNET | 访问网络 | 加载线上资源、调用服务端接口 |
| ACCESS_NETWORK_STATE | 查看网络状态 | 检测网络连接情况，优化请求策略 |
| ACCESS_WIFI_STATE | 查看 Wi-Fi 状态 | 检测 Wi-Fi 连接情况 |
| WAKE_LOCK | 保持唤醒 | 确保推送消息能够正常到达 |
| POST_NOTIFICATIONS | 发送通知 | 向用户发送推送通知（Android 13+ 需单独申请） |
| JPUSH_MESSAGE | 极光推送消息接收 | 接收极光推送下发的消息 |
| VIBRATE | 控制振动器 | 推送到达时提供振动提醒 |
| RECEIVE_USER_PRESENT | 接收解锁广播 | 极光推送 SDK 依赖，用于提升推送到达率 |

上述权限均会在用户安装本应用时通过系统提示或应用权限管理机制告知用户，用户可根据需要选择开启或关闭。关闭部分权限可能导致相应功能无法正常使用。

## 三、我们如何使用信息

我们收集的信息将用于以下目的：

1. **提供核心服务**：根据用户输入生成、扩展、编辑思维导图，管理工作区和文件。
2. **AI 对话处理**：将用户对话内容发送至 AI 模型，以生成结构化的思维导图内容或回复。
3. **安全保障**：识别、预防和应对欺诈、滥用、安全漏洞、技术故障等问题，包括基于 IP 地址的限流和封禁管理。
4. **产品优化**：通过匿名化的埋点数据分析用户使用习惯，改进功能设计、交互体验和性能表现。
5. **消息推送**：通过极光推送向用户发送服务通知、重要更新、协作提醒等。
6. **客户服务**：处理用户的反馈、建议和投诉。
7. **法律法规要求**：遵守适用的法律法规、监管要求或司法程序。

## 四、第三方 SDK 及服务

为保障部分功能的实现，我们接入了第三方 SDK 或服务，其可能收集用户的部分信息：

| SDK/服务名称 | 服务商 | 使用目的 | 可能收集的信息 |
|-------------|--------|---------|---------------|
| 极光推送 JPush | 深圳市和讯华谷信息技术有限公司 | 消息推送 | 设备标识（Registration ID）、设备型号、操作系统版本、APP 版本、网络信息、位置信息（用于推送通道优化） |
| 第三方 AI 大模型服务 | 由用户选择或运营方内置配置 | AI 内容生成 | 用户对话内容、思维导图上下文 |

第三方 SDK 的隐私政策由相应服务商提供，用户可在其官网查阅。我们建议用户在使用相关功能前仔细阅读上述第三方的隐私政策。

## 五、信息共享

除以下情形外，我们不会向任何第三方共享用户的个人信息：

1. **用户同意**：获得用户的明确同意后，向第三方共享。
2. **服务提供需要**：向第三方 AI 服务提供商传输必要的对话内容，以生成 AI 回复或思维导图。
3. **技术服务提供**：向极光推送等第三方 SDK 提供商共享必要的设备信息和推送标识，以实现消息推送功能。
4. **法律法规要求**：根据法律法规、法院命令、政府要求或司法程序，向有权机关提供必要信息。
5. **权益保护**：为预防和解决欺诈、安全威胁、知识产权侵权或其他违法活动，在必要范围内共享信息。

我们不会将用户的个人信息出售、出租或以其他方式提供给第三方用于其独立的市场营销目的。

## 六、信息存储与安全

### 6.1 存储地点

用户的思维导图内容、对话记录、工作区信息、埋点数据等将存储于运营方控制的服务器上，服务器位于中国大陆境内。

用户的访客 ID、访客签名密钥、当前工作区 ID、用户自定义 AI 模型配置（含 API Key）等信息存储在用户设备的本地存储（localStorage）中。

### 6.2 存储期限

- 思维导图、对话记录、工作区数据：自用户最后一次操作起保留，直至用户删除或主动要求注销/清除数据。
- 埋点数据：保留不超过 12 个月。
- IP 历史记录：最多保留最近 10 条。
- 反馈信息：保留至问题处理完成或法律法规要求的期限。

### 6.3 安全措施

我们采取合理的技术和管理措施保护用户个人信息的安全，包括但不限于：

- 对服务端接口采用 HTTPS 加密传输。
- 对生产环境请求进行 HMAC-SHA256 签名鉴权。
- 对数据库访问进行权限控制和审计。
- 对用户输入进行 XSS 过滤等安全处理。

但互联网传输并非绝对安全，我们无法保证信息在传输过程中不被拦截。用户应妥善保管自己的设备和本地存储信息，尤其是自行配置的第三方 AI API Key。

## 七、用户权利

根据适用的法律法规，用户对自己的个人信息享有以下权利：

1. **查阅和复制**：用户有权查阅和复制我们收集的其个人信息。
2. **更正和补充**：如用户发现个人信息有误，有权要求我们更正或补充。
3. **删除**：用户可主动删除其创建的思维导图、工作区及上传的文件。在符合法律规定的情形下，用户有权要求我们删除其个人信息。
4. **撤回同意**：用户可通过以下方式撤回对部分信息收集的同意：
   - 在设备系统设置中关闭推送通知权限，以撤回对消息推送的同意；
   - 在设备系统设置中清除应用数据或卸载本应用，以停止本地标识信息的收集；
   - 通过本政策末尾的联系方式联系我们，要求停止使用其个人信息。
5. **注销**：当前版本暂未提供自助注销入口，用户可联系运营方处理注销及数据清除请求。注销后，除法律法规另有要求外，我们将删除或匿名化用户的个人信息。

用户行使上述权利时，可通过本政策末尾的联系方式与我们联系。我们将在收到请求后的合理期限内予以响应。

## 八、未成年人保护

本应用主要面向成年人用户。若用户为未满 18 周岁的未成年人，应在监护人指导下阅读本政策并在监护人同意的前提下使用本应用。

我们不会主动识别用户年龄，也不会主动收集未成年人的个人信息。如果我们发现自己无意中收集了未成年人的个人信息，将尽快删除相关内容。

## 九、Cookie 与类似技术

本应用目前未使用 Cookie 技术。部分本地存储功能通过浏览器/ WebView 的 localStorage 实现，用于保存用户的访客身份、界面设置、AI 模型配置等本地偏好信息。
