# Checklist

> 对应 spec.md：非线性对话核心闭环

---

## 延伸方向按钮

- [ ] `server/src/config/prompts.ts` 中 system prompt 稳定要求 AI 输出 `🌱 延伸方向：` 格式
- [ ] 前端能从 AI 回答中正确解析延伸方向列表
- [ ] 延伸方向渲染为 2-4 个可点击按钮
- [ ] 原始 `🌱 延伸方向：` 文本从回答正文中移除
- [ ] 点击按钮后自动创建子节点
- [ ] 点击按钮后自动切换到子节点
- [ ] 点击按钮后自动发送问题给 AI
- [ ] AI 回答不包含延伸方向时不显示按钮
- [ ] 延伸方向点击事件已埋点（`extension_direction_click`）

## 节点自动摘要

- [ ] Node 数据模型包含 `summary?: string` 字段
- [ ] `summary` 字段能持久化到 MongoDB 并正确读取
- [ ] 用户可手动点击"生成摘要"按钮
- [ ] 手动触发后调用 AI 生成摘要（复用 `CONCLUSION_EXTRACTION_PROMPT`）
- [ ] 摘要存入节点 `summary` 字段
- [ ] 画布节点卡片展示摘要预览（可折叠）
- [ ] 导航树节点悬停展示摘要
- [ ] 聊天面板节点信息区展示完整摘要
- [ ] 支线结束检测逻辑正常工作（轮数 + 时间）
- [ ] 自动触发时提示用户确认，用户可忽略
- [ ] "回到上一级"时如果子节点有未摘要新增对话，提示是否生成摘要
- [ ] 摘要生成事件已埋点（`summary_generated`）
