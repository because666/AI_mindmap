贝塞尔曲线 + 流光渐变效果演示

三种真正的流光渐变实现方案，光线自然流动，带有渐变过渡和发光感

方案A: SVG渐变动画
使用 SVG <linearGradient> + <animate> 实现渐变沿路径自然流动，光线有真实的渐变过渡
方案B: 光线扫过效果
使用 stroke-dasharray + 多色渐变，模拟一束光线沿路径扫过的效果，带有拖尾发光
方案C: 自然流光（推荐）
多条光线以不同速度流动，产生自然的"能量传输"效果，最接近真实的流光渐变
方案D: 多节点网络流光
模拟真实思维导图场景，多条贝塞尔曲线配合不同颜色的自然流光，每条边独立流动
方案E: 呼吸式流光
光线亮度周期性变化，像呼吸一样明暗交替，配合缓慢流动，营造生命感
方案F: 最终推荐（综合最优）
Bezier曲线 + 多层SVG发光滤镜 + 自然渐变流光 + 微妙呼吸节奏，最适合DeepMindMap
