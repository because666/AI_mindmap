## 完整配音文字（可直接复制到剪映）

```
和AI聊复杂问题，聊几轮就忘了在问啥。

所以我跟AI说：帮我做一个对话自动变成思维导图的工具。

AI写代码，我提需求，调试，部署，上线。

现在它长这样，免费的，链接评论区。
```

**总字数：** 约55字（不含镜头1的文字叠加）
**总时长：** 约27秒配音 + 3秒无声文字 = 30秒
**语速：** 偏快，口语化

---

## 语音生成步骤

1. 打开剪映，导入录屏素材
2. 点击「文字」→「文本朗读」
3. 粘贴上方配音文字
4. 音色选择：**不要选播音腔**，选偏口语化的音色（如"活力男声""元气女声"）
5. 语速：1.1x（比正常稍快，跟上快剪节奏）
6. 生成后对齐到对应镜头
