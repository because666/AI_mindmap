## 二、第一阶段规划中功能（2-6 周）

### 2.1 非线性对话体验（P0/P1）

| 功能 | 说明 | 优先级 |
|---|---|---|
| 延伸方向按钮 | AI 回答后显示 2-4 个可点击延伸方向，点击自动创建子节点并进入 | P0 |
| 节点自动摘要 | 支线聊完后自动提炼节点摘要，展示在画布与导航树 | P1 |
| 节点导航树优化 | 高亮当前路径、增加“回到上一级”、自动展开当前节点 | P1 |
| 输入框智能分叉提示 | 检测用户追问子主题时，提示创建分支 | P1 |

### 2.2 留存与增长（P1）

| 功能 | 说明 | 优先级 |
|---|---|---|
| 模板库 | 系统预置地图模板，新用户首次进入可一键使用 | P1 |
| 工作区地图库 | 在工作区内管理所有历史地图，支持搜索与排序 | P1 |
| 管理员推荐工作区 | 用户可见管理员置顶工作区，可一键复制到自己空间 | P1 |
| 首次进入空工作区体验 | 不展示空白画布，优先引导选择模板 | P1 |

### 2.3 数据与运营基础设施（P1）

| 功能 | 说明 | 优先级 |
|---|---|---|
| 用户行为埋点 | 主服务接收事件，Admin Dashboard 展示 | P1 |
| 后台 AI 模型池 | 管理员可配置多模型、默认模型、fallback、按任务路由 | P1 |
| 后台数据洞察中心 | Dashboard 2.0、用户漏斗、留存、功能采用矩阵、系统健康 | P1 |

## 四、功能优先级说明

- **P0**：核心啊哈时刻，必须尽快实现
- **P1**：支撑留存、运营、数据洞察，紧随 P0
- **P2**：探索性方向，验证后再深入
- **P3**：长期愿景

当前第一阶段聚焦：

```text
延伸方向按钮 → 节点摘要 → 导航树优化 → 模板库 → 工作区地图库 → 埋点 → 后台模型池 → 数据洞察
```
